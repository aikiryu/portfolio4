1. 변수 
데이터를 저장하는 공간
let varName = data;
하나의 변수에는 하나의 데이터만 저장된다. 

2. 데이터의 타입
Number 숫자형 : 123,456,0.123
String 문자열 : "안녕하세요", "반갑습니다"
Boolean 논리형 : true, false; 
Undefined, null 비어있음 

3. 기본연산자
+ 더하기 : 문자와 다른 타입의 데이터를 더할 경우
         하나의 문자열로 이어진다.
- 빼기 
* 곱하기
/ 나누기
% 나머지 : 자연수/자연수를 했을 때 자연수로
  더 이상 나눠지지 않는 남은 수를 나머지라고 부른다.
